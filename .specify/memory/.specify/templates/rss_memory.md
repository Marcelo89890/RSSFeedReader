# Plantilla de memoria: RSS Feed

Tipo: `feed`

Campos recomendados:
- title: Título del feed
- url: URL del feed RSS
- last_checked: Fecha/Hora de la última comprobación
- items: Lista de artículos (id, title, link, published)

Ejemplo:
```
- title: "Mi Feed"
- url: "https://example.com/rss"
- last_checked: "2026-01-30T12:00:00Z"
- items: []
```
